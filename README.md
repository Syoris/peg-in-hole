# peg-in-hole

## Installation
- Add `VORTEX_INSTALLATION_PATH` to your user env. variable (i.e. `C:\CM Labs\Vortex Studio 2023.8`)
- `poetry install`
- `poetry shell`
- `pip install tensorflow`

Note:
- To have `.venv` in projec: `poetry config virtualenvs.in-project true`
